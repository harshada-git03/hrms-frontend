# NexHR — Frontend

A modern HRMS frontend built with React 18 + Vite + TailwindCSS.

## Tech Stack
- **Framework:** React 18 + TypeScript + Vite
- **Styling:** TailwindCSS + shadcn/ui
- **Animations:** Framer Motion
- **State:** Zustand + React Query
- **HTTP:** Axios
- **Deploy:** Vercel

## Pages
- 🔐 Login
- 📊 Dashboard (live stats)
- ⏰ Attendance (clock in/out + charts)
- 📅 Leave Management
- 👥 Employee Directory
- 💰 Payroll & Payslips
- 📢 Announcements + Indian Holidays
- 📄 Documents & E-sign
- 🤖 AI Assistant (Gemini)
